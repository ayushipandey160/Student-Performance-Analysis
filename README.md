# Student Performance Analysis

## Internship Task 1

This project analyzes the Student Performance Dataset using Python.

### Tasks Performed

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Average Grade Calculation
- Correlation Analysis
- Gender-wise Performance Analysis
- Data Visualization

### Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

### Visualizations

- Histogram of Final Grades
- Study Time vs Grade Scatter Plot
- Gender-wise Average Grade Bar Chart
